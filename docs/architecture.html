log-anomaly-detector Architecture

Synthetic security log pipeline → feature extraction → unsupervised anomaly detection

Log Generator

  • • SSH auth logs — brute-force patterns, failed passwords
  • • Web server logs — Apache Combined Format, suspicious paths
  • • System logs — OOM, AppArmor, critical errors
  • • Configurable anomaly ratio (default 5%)

Feature Extractor

  • • Type-specific regex parsers per log format
  • • 5-minute time window aggregation
  • • 28+ features: event counts, error rates, IP entropy
  • • Time features: hour of day, weekday, business hours

Anomaly Detector

  • • Isolation Forest (primary) — 200 trees
  • • Autoencoder (optional) — PyTorch reconstruction error
  • • Rich CLI output with anomaly tables and scores
  • • Pattern classification: brute-force, scanning, exploits
